Pengkarir - Temukan karirmu bersama Berkarir

Project information

  • Company: PT. Kompas Ion Teknologi
  • Category: Job Platform
  • Project start: June 1st, 2019
  • Play store URL: Google Play Store

Pengkarir is job platform application that mean for user as company that seek employee candidate. Build by myself as employee at PT. Kompas Ion Teknologi from begining until October 2021. First Pengkarir build with Java and MVP Design Pattern, then on July 2020 decided to use Kotlin and used MVVM Design Pattern. Berkarir have Features that can Create Company Information, Search Applicant, Create Jobs, Chat, Invite interview, and Status Confirmation.

Library used :
- Firebase Authentication
- Firebase Crashlytich
- Firebase Realtime Database
- Firebase Cloud Messaging
- Easy Permission
- Room
- Retrofit
- Picasso
- Stfalcon ImageViewer
- Ucrop

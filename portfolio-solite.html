Solite - Dimsum POS

Project information

  • Category: Point Of Sales (POS) system
  • Project start: September 1st, 2020
  • Play store URL: Google Play Store
  • Github URL: Github Repository

Solite is app base Point Of Sales System build with Kotlin and MVVM Design Pattern that have Login, Order, Buy, Order History, Sales Report, Master Data, Bluetooth Print, and Online Offline Data Storage.

Library used :
- Firebase Firestore
- Firebase Authentication
- Easy Permission
- Picasso
- Retrofit
- Room
- Ucrop
